# Diplom_2

# Задание 2: тестирование API Stellar Burgers

Нужно протестировать ручки API для Stellar Burgers.

# Создание пользователя:
(тесты расположены в файле 'test_registration')
- создать уникального пользователя;
- создать пользователя, который уже зарегистрирован;
- создать пользователя и не заполнить одно из обязательных полей.
# Логин пользователя:
(тесты расположены в файле 'test_user_login')
- логин под существующим пользователем,
- логин с неверным логином и паролем.

# Создание заказа:
(тесты расположены в файле 'test_order_create')
- с авторизацией,
- без авторизации,
- с ингредиентами,
- без ингредиентов,
- с неверным хешем ингредиентов.


**Установка зависимостей**

> `$ pip install -r requirements.txt`

**Запуск всех тестов одной командой**

>  `pytest -v`

**Allure-отчет о тестировании в формате веб-страницы**

>  `allure serve allure_results`
